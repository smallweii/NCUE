MACHINE LANGUAGES AND ASSEMBLY LANGUAGES ARE LOW LEVEL LANGUAGES. C LANGUAGES, C++ LANGUAGES, AND JAVA LANGUAGES ARE HIGH LEVEL LANGUAGES.  LOW LEVEL LANGUAGES ARE CLOSER TO THE HARDWARE THAN ARE HIGH LEVEL LANGUAGES. LOW LEVEL LANGUAGES CAN BE CONVERTED TO MACHINE CODE WITHOUT USING A COMPILER OR INTERPRETER, AND THE RESULTING CODE RUNS DIRECTLY ON THE ROCESSOR. LOW LEVEL LANGUAGES ARE SIMPLE, BUT ARE DIFFICULT TO USE.
